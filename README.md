# namearth5005-plugins

Claude Code plugins by namearth5005.

## Install

```
/plugin marketplace add namearth5005/namearth5005-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **remindctl-mcp** | Apple Reminders integration via remindctl CLI |

## Install a Plugin

```
/plugin install remindctl-mcp@namearth5005-plugins
```
